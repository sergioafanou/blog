Introdução

Desenvolvido para workflow de ciência de dados e aprendizagem de máquina, o Anaconda é um gerenciador de pacotes, gerenciador de ambiente e uma distribuição das linguagens Python e R de código aberto.

Este tutorial ensina a como instalar o Anaconda em um servidor Ubuntu 18.04. Para uma versão mais detalhada deste tutorial, contendo explicações mais completas para cada etapa, consulte Como instalar a distribuição de Python Anaconda no Ubuntu 18.04.

Passo 1 — Obtenha a versão mais recente do Anaconda

Por um navegador da Web, vá à página da Distribuição Anaconda, disponível no link a seguir:

https://www.anaconda.com/distribution/

Encontre a versão mais recente do Linux e copie o script bash de instalação.

Passo 2 — Faça o download do script bash do Anaconda

Uma vez logado no seu servidor Ubuntu 18.04 como um usuário sudo sem root, vá ao diretório /tmp e use curl para fazer o download do link copiado do site do Anaconda.

  • cd /tmp

  • curl -O https://repo.anaconda.com/archive/Anaconda3-2019.03-Linux-x86_64.sh

Passo 3 — Confira a integridade de dados do instalador

Verifique a integridade do instalador com a verificação de hash criptográfica usando o checksum SHA-256:

  • sha256sum Anaconda3-2019.03-Linux-x86_64.sh

Output
45c851b7497cc14d5ca060064394569f724b67d9b5f98a926ed49b834a6bb73a  Anaconda3-2019.03-Linux-x86_64.sh

Passo 4 — Execute o script do Anaconda

  • bash Anaconda3-2019.03-Linux-x86_64.sh

Você receberá a seguinte saída para revisar o acordo de licença. Pressione ENTER até o final.

Output

Welcome to Anaconda3 2019.03

In order to continue the installation process, please review the license
agreement.
Please, press ENTER to continue
>>>
...
Do you approve the license terms? [yes|no]

Ao final da licença, digite yes para concordar com a licença e concluir a instalação.

Passo 5 — Finalize o processo de instalação

Depois de concordar com a licença, você será solicitado a escolher o local da instalação. Pressione ENTER para aceitar o local padrão ou especifique um local diferente.

Output
Anaconda3 will now be installed into this location:
/home/sammy/anaconda3

  - Press ENTER to confirm the location
  - Press CTRL-C to abort the installation
  - Or specify a different location below

[/home/sammy/anaconda3] >>>

A partir disso, a instalação prosseguirá. Lembre-se de que o processo de instalação pode levar algum tempo.

Passo 6 — Selecione opções

Quando a instalação for concluída, você receberá a seguinte saída:

Output
...
installation finished.
Do you wish the installer to prepend the Anaconda3 install location
to PATH in your /home/sammy/.bashrc ? [yes|no]
[no] >>>

É recomendável digitar yes para usar o comando conda.

Passo 7 — Ative a instalação

Agora é possível ativar a instalação com o seguinte comando:

  • source ~/.bashrc

Passo 8 — Teste a instalação

Use o comando conda para testar a instalação e ativação:

  • conda list

Você receberá a saída de todos os pacotes disponíveis pela instalação do Anaconda.

Passo 9 — Configure os ambientes do Anaconda

É possível criar ambientes Anaconda com o comando conda create. Por exemplo, um ambiente Python 3 chamado my_env pode ser criado com o comando a seguir:

  • conda create --name my_env python=3

Ative o novo ambiente dessa forma:

  • conda activate my_env

Seu prefixo de prompt de comando será alterado quando você estiver em um ambiente Anaconda ativo. Agora, tudo está pronto para que você comece a trabalhar em um projeto.
